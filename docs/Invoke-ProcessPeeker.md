# Invoke-ProcessPeeker

## SYNOPSIS

A TUI process viewer.

## SYNTAX

```yaml
Invoke-ProcessPeeker [<CommonParameters>]
```

## DESCRIPTION

Launch a TUI that can be used to display process information. Select a process from the left-hand list to display process details in the right-hand pane.

This command has an alias of ProcessPeeker.

## EXAMPLES

### Example 1

```powershell
PS C:\> Invoke-ProcessPeeker
```

This command has an alias of ProcessPeeker.
