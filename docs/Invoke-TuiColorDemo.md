# Invoke-TuiColorDemo

## SYNOPSIS

Run the TUI color demo.

## SYNTAX

```yaml
Invoke-TuiColorDemo [<CommonParameters>]
```

## DESCRIPTION

This function will display a simple TUI that allows you to see the various color combinations for the main window. Select the foreground and background colors, then click the Try It button. You can use Reset to return to the default colors.

The TUI also includes a small code snippet that shows how to set the colors in your own TUI. You can't edit the code in the TUI, but you should be able to copy and paste it.

## EXAMPLES

### Example 1

```powershell
PS C:\> Invoke-TuiColorDemo
```

This command has an alias of TuiColorDemo
